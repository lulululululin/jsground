# 如何增加日志

手动修改data里的js文件. 
也可以增加或替换 `./data/20nnnnx.js` 文件. 我个人计划是每个月换一个. 有点像朋友圈, 展示最近这个月的内容. 更换的时候修改index.html里12行加载js文件路径.
